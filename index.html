Sobre Rick and Morty

Rick and Morty é uma série animada que acompanha as aventuras de Rick Sanchez,
um cientista brilhante, excêntrico e alcoólatra, e seu neto Morty, um jovem inseguro.

Este projeto consome uma API pública de Rick and Morty, permitindo buscar
personagens e exibir informações como nome, status e espécie.
